Introduction to Javascript

Javascript is an interprated language hence we need an interpretor to execute js code

The browser comes with built in JS Engine(has an interpretor) that translates Javascript to machine code , feeds the processor and gets back feedback/result of the execution.

We can also download other Javascript runtimes/translators/interpretors like Node Js , Deno Js or Bun Js to execute Javascript outside the browser.

JS on the browser

To use JS on the browser, create a html file with a script element.

The script element can contain javascript code in it or a src(source) attribute with the url path to a javascript(.js) file.
